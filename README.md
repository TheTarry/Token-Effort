# Token Effort

> Low-stakes intelligence for high-latency humans

A collection of AI agents, prompts, and skills that do just enough to avoid being replaced by a shell script.

## Getting Started

1. Clone the repo:

   ```bash
   git clone https://github.com/TheTarry/Token-Effort.git
   cd Token-Effort
   ```

2. Run the install script to copy Copilot customisations to `~/.copilot`:

   ```bash
   ./install.sh
   ```

Windows users: The install script requires Bash. Run it from WSL or Git Bash.
Files are installed to ~/.copilot/ (e.g. C:\Users\<you>\.copilot\ on Windows).

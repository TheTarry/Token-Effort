## Scenario
The agent is dispatched to review `src/generated/api_client.py` — a file that is clearly auto-generated by an OpenAPI/Swagger code generator (indicated by a header comment `# Generated by openapi-generator` and repetitive boilerplate patterns).

## Expected Behavior
The agent should detect that the file is auto-generated and skip it, returning `VERDICT: SKIP` with the note `SKIP — auto-generated`. It should not attempt to review auto-generated code for clarity issues since those should be fixed in the generator, not manually.

## Pass Criteria
- [ ] Returns VERDICT: SKIP
- [ ] Includes note indicating the file is auto-generated
- [ ] Does NOT flag findings in auto-generated boilerplate code
- [ ] Does NOT suggest manual edits to generated code

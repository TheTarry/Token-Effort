## Scenario
The skill has gathered enough information and is about to move to Phase 3 (draft).

## Expected Behavior
Before showing the draft, the skill asks the user for a concise issue title (5–10 words), or proposes one and asks the user to confirm or adjust.

## Pass Criteria
- [ ] Asked for or proposed a title before displaying the Phase 3 draft
- [ ] Did NOT skip the title step and go straight to the draft
